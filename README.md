# Latihan Bahasa Jepang Online

Folder ini siap di-upload ke GitHub Pages.

## Isi folder

- `index.html` adalah halaman menu utama.
- `exercises/1-kanji-yomi.html` adalah latihan `1. 漢字読み`.
- `.nojekyll` membantu GitHub Pages menyajikan file statis apa adanya.

## Cara menambah latihan berikutnya

1. Simpan latihan baru sebagai file HTML di folder `exercises`.
2. Gunakan nama file yang aman untuk web, misalnya `2-kotoba.html`.
3. Tambahkan link baru di `index.html` pada bagian `Daftar latihan`.

## Cara publish singkat

1. Buat akun GitHub.
2. Buat repository baru, misalnya `latihan-bahasa-jepang`.
3. Upload semua isi folder `online-site` ke repository.
4. Buka `Settings` -> `Pages`.
5. Pilih `Deploy from a branch`, branch `main`, folder `/root`.
6. Simpan, tunggu beberapa menit, lalu buka link GitHub Pages yang muncul.
